01. 시스템 개요 및 개발 배경

  • 해운 산업의 디지털 전환(DX) 선도 및 환경 보호 기여
  • 국제해사기구(IMO) CII 규제 강화에 따른 신속 대응 인프라
  • 선종별/연료별 최적 운항 속도 도출 및 탄소 배출 실시간 정밀 연산
  • Python 엔진 탑재로 신뢰성 높은 환경 성과 데이터 지원

02. EcoShip-Analyzer 핵심 기능

📊 실시간 시뮬레이션

사용자가 컨트롤 패널을 제어하는 즉시 속도, 배수량, 선종 및 연료 변화에 연동하여 이산화탄소 예측 그래프와 수치 반영

🏅 IMO CII 등급 평가

IMO 친환경 가이드라인에 근거한 연간 배출 한계 대비 현재 상태를 분석하고 탄소 집약도 기준의 종합 A~E 등급 판정

📋 운항 기록 관리

Streamlit 세션 상태(Session State) 기반 축적 기술로 시뮬레이션 기록을 이력으로 관리하고 실시간 트렌드 피드백

03. 엔지니어링 연산 엔진: 해군제수법

Admiralty Coefficient 기반 소요 제동 마력(BHP) 추정 연산 엔진

C = Δ23 × V3 BHP
  • Δ (Displacement): 선박의 현재 수하 상태를 반영하는 만재 배수량 (ton)
  • V (Speed): 마력 수요의 비약적 변동을 유발하는 선박 운항 속도 (knots)
  • BHP (Brake Horsepower): 선체를 밀어내는 엔진 실질적 요구 출력 (kW) [속도의 세제곱에 정비례]

05. 운항 조건 시뮬레이션 UI 설계

  • 사용자 입력 즉시 실시간 변환 피드백 구조 설계
  • 설계 한계 속도 도달 시 UI 경고 메시지 출력 가이드라인 마련
  • 연료 종류별 배출 계수(Cf) 연동 모델 실시간 반영
  • 직관적인 Metric 카드 구성으로 5대 연산 가독화 완성
